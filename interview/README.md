### 一、基础理论   
#### 1、说一下你对 promise 的理解。（阿里一面、网易一面）    
promise 是对 js 异步回调的一个封装，通过 then 方法链式调用的方式，避免了产生传统的回调地狱，不仅改善了自己的编码体验，还极大提高了代码的可维护性，使代码看起来更加的规范整洁。
